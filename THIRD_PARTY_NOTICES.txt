6. プロジェクト専用Tesseract CLI
--------------------------------------------------

pytesseractから上記Tesseractエンジンを呼び出すためのこのアプリ固有のCLIです。
ソースを同梱し、Windows全体へのTesseractインストールは行いません。

12. 同梱していないランタイム
--------------------------------------------------
Tesseract同梱版は Microsoft .NET Framework 4.7以上と Microsoft Visual C++
2015-2022 x64 Runtime を必要とします。これらMicrosoft製ランタイムは本フォルダに
複製・同梱していません。不足するPCではMicrosoft公式配布物を利用者側で導入してください。

不採用ビルドについて
--------------------------------------------------
UB Mannheim配布の tesseract-ocr-w64-setup-5.4.0.20240606.exe は調査のみ行い、
そのビルドがGPL-2.0の libjbig-0.dll へ直接依存することを確認したため、
非公開販売アプリには採用せず、その実行ファイル・DLLを同梱していません。
